search mode, move to **f**irst character, move un<strong>t</strong>il you find

!SLIDE normal
## Search Based Movement

    f t F T     " one line, forward / BACKWARD
    / ?         " whole file, forward / backward

While searching

    ;           " one line find again
    n           " next
    N           " previous
